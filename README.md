# ✅ To-Do App

A simple and clean To-Do app built using **React.js** — perfect for managing your daily tasks and practicing frontend skills! 🚀

---

## 🔑 Key Features

1. 📝 **Add Tasks** – Type and add your daily goals.
2. 📋 **View List** – See all your tasks at one glance.
3. ✅ **Mark as Done** – Click to check off completed items.
4. 🗑️ **Delete Tasks** – Remove tasks you don't need anymore.
5. 💾 **State Management** – Uses React hooks (`useState`) for state.
6. ⚡ **Lightweight UI** – Beginner-friendly with no UI libraries.
7. 🌐 **Responsive Design** – Works on mobile & desktop.
8. 🧠 **Clean Code** – Easy to read, beginner-friendly structure.

---

## 🛠 Tech Stack

- ⚛️ React.js (Frontend)
- 🌐 HTML, CSS, JavaScript
